Kotlin with Android Summary =>
1) Android Studio version >= 3.0 or install Kotlin plugin in AS version <3.0 (File -> Settings -> Plugins -> Install JetBrains -> search Kotlin)
2) We can convert existing java file kt file by ctl+shift+alt+k
3) Need to verify gradle file -> Configure Kotlin with Gradle -> Kotlin compiler and runtime version (1.2.71 -- 1.1.4-2), newly released is 1.3
4) Work with most popular frameworks -> Dagger, Butterknife, Data Binding, DBFlow, etc... (later or initially we can start with these)
5) We can mix Java and Kotlin in one project. Same way Kotlin is with iOS, NativeScript, JavaScript, Python, many more...
